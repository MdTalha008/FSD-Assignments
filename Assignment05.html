What are localStorage and sessionStorage?

localStorage and sessionStorage are web storage mechanisms in HTML5 that allow websites to store data in the browser.

  • localStorage: Stores data with no expiration time. The data is saved across browser sessions and only cleared when explicitly deleted by the user or website.
  • sessionStorage: Stores data for the duration of the page session. The data is deleted when the browser tab is closed.

These are commonly used to save user preferences, form data, or temporary states without requiring a server.
